# Бот для перевізників (канал + бот) — v2

Повторює схему `MK Merchants`: оголошення публікуються в каналі з кнопкою,
перевізники клікають → бот у приватному чаті приймає два тарифи (з ПДВ / без ПДВ),
адмін отримує звіт з найкращою ціною.

---

## Що нового у v2

- **Декілька адміністраторів.** Є головний адмін (з `ADMIN_ID` у середовищі) і динамічний список у БД — додавати / видаляти прямо з бота.
- **Усі команди через кнопки.** Внизу чату постійне меню, під оголошеннями — inline-кнопки (звіт / закрити / відкрити). Команди `/new`, `/list`, `/report`, `/close`, `/reopen` теж працюють, але ними вже можна не користуватись.
- **Номери оголошень монотонно зростають** (1, 2, 3 …) — навіть після закриття чи видалення (SQLite `AUTOINCREMENT`).
- **Клікабельні посилання на профілі перевізників** у звітах і сповіщеннях (HTML `<a href="https://t.me/username">…</a>`, з fallback на `tg://user?id=…`).

## Функціонал

**Головний адмін (ваш `ADMIN_ID`) і додані адміни:**

- **🆕 Нове оголошення** — майстер (маршрут, вантаж, вага, дата, контакт). Пост публікується у каналі з inline-кнопкою «💵 Ввести вашу пропозицію».
- **📋 Активні оголошення** — список 🟢/🟡 з кнопками «📊 Звіт», «🔴 Закрити», «🟢 Відкрити».
- **📊 Звіт** — усі пропозиції: найкраща ціна з ПДВ і без ПДВ окремо, посилання на профілі, телефони.
- Миттєве сповіщення щоразу, коли перевізник вводить ціну (з посиланням на його Telegram).

**Тільки головний адмін:**

- **👥 Адміни** — список з кнопками «➕ Додати», «🗑 Видалити».
- Додавання: можна переслати повідомлення від людини АБО ввести її user ID. Новому адміну приходить вітання від бота.

**Перевізник:**

- Клік у каналі → картка заявки в приватному чаті з ботом.
- Кнопки: «📝 Ввести тариф з ПДВ», «📝 Ввести тариф без ПДВ», «📱 Передати контакт».
- Може оновлювати ціни будь-яку кількість разів — останнє значення йде у звіт.

**Автоматично:**

- 🟢 → 🟡, коли надходить перша пропозиція.
- `request_id` унікальний — перевізник завжди бачить одну й ту саму заявку для конкретного оголошення.

## Деплой на Railway

### 1. Передумови

- [x] Канал створено, бот доданий як адміністратор з правом «Публікувати повідомлення».
- [x] Токен з `@BotFather`.
- [x] `ADMIN_ID` — ваш Telegram ID (через `@userinfobot`).
- [x] `CHANNEL_ID` каналу (через `@username_to_id_bot`, починається з `-100`).
- [x] `BOT_USERNAME` — юзернейм бота без `@`.

### 2. Завантажити код на GitHub

1. Створити приватний репо на [github.com](https://github.com) (наприклад, `agronorma`).
2. Завантажити файли: «Add file → Upload files» або через git.

`.env` у репозиторій не додається — він у `.gitignore`.

### 3. Розгорнути на Railway

1. [railway.app](https://railway.app) → вхід через GitHub.
2. **New Project → Deploy from GitHub repo →** оберіть репо.
3. **Variables** — додати:

   | Key | Value |
   |---|---|
   | `BOT_TOKEN` | токен з @BotFather |
   | `ADMIN_ID` | `772817` |
   | `CHANNEL_ID` | `-1003958283733` |
   | `BOT_USERNAME` | `vzayemodiya_z_pereviznykom_bot` |
   | `DB_PATH` | `/data/bot.db` |

4. **Volumes → + New Volume →** Mount path `/data` (щоб БД не обнулялась після деплою).
5. **Deploy.** У логах має з'явитись `Старт бота…`.

### 4. Перевірка

1. `/start` боту з вашого акаунта → побачите адмін-меню.
2. Кнопка **🆕 Нове оголошення** → пройти майстер → перевірити, що пост з'явився в каналі.
3. З іншого акаунта клікнути кнопку під постом → ввести тариф.
4. Вам має прийти сповіщення з посиланням на перевізника.
5. **📋 Активні оголошення → 📊 Звіт** — побачите повний звіт.

## Додавання адміністраторів

Два способи — обидва лише для головного адміна:

1. **Через бот** (рекомендую):
   - `👥 Адміни` → `➕ Додати адміна` → переслати будь-яке повідомлення від людини (або написати її user ID).
   - Бот одразу спробує написати новому адміну привітання, щоб він бачив меню.
2. **Вручну** командою: `/addadmin <user_id>` у приватному чаті з ботом.

Видалити: `👥 Адміни` → тицнути `🗑 Ім'я` біля потрібного, або `/rmadmin <user_id>`.

Головного адміна видалити з бота не можна — він задається через `ADMIN_ID` у Variables.

## Локальний запуск

```bash
cd pereviznyk-bot
python3 -m venv .venv
source .venv/bin/activate        # Windows: .venv\Scripts\activate
pip install -r requirements.txt
cp .env.example .env
# впишіть BOT_TOKEN тощо у .env
export $(grep -v '^#' .env | xargs)      # Linux/Mac
python bot.py
```
